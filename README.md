## Description
The server-side core of the Patient-Staff System, This backend acts as a real-time bridge, relaying patient form data to staff monitors.

Tech Stack: Node.js + TypeScript + Socket.io (WebSockets) + Express

## Getting Started
Installation

```bash
npm install
```

Environment Variables : Create a .env file in the root folder:

```bash
NEXT_PUBLIC_SERVER_URL=https://your-backend-url.com // or http://localhost:3001
```

## Development mode

```bash
npm run dev
```

## Production mode

```bash
npm start
```
